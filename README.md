# Triple-R Service Co: 3D Models

This repository holds solid models for tools and parts that I've made for proprietary dealer tools, unobtainium parts, and other odds and ends in the course of running [Triple-R Service Co](https://youtube.com/@rrrserviceco). Feel free to use them to make the parts for yourself, or modify them as you see fit! Open hardware is the way :)

## Tools

* **John Deere clutch alignment tool JDG693** [[Onshape](https://cad.onshape.com/documents/0ad82c9d7919dc6fb16590a8/w/a66f2ec2fa3fc32d39691eec/e/9deade055c3da2cde84fd22b?renderMode=0&uiState=69a7c8798d7edbfe8572f589)]: a clutch alignment tool. I bought one for working on a John Deere 790 compact tractor, but I'm sure it's used on other JD clutches. Costs $250 from greenpartstore.com, and after ordering from them I discovered it's only $60 from Servicegard...but it should only be like $10 to 3D print!
